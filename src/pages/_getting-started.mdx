# Unlocking the Power of GIS at RCC

The Research Computing Center (RCC) at the University of Chicago offers a robust suite of Geospatial Information Systems (GIS) tools integrated with our high-performance computing (HPC) infrastructure. This powerful combination enables researchers to tackle complex spatial analyses, process massive geospatial datasets, and harness cutting-edge GIS technologies.

## GIS on High-Performance Computing

Our HPC environment is optimized to support a wide array of GIS software platforms:

| Software | Description |
|----------|-------------|
| GRASS GIS | Open-source GIS for geospatial data management and analysis |
| QGIS | User-friendly, open-source GIS for viewing, editing, and analyzing data |
| ArcPy | Python site-package for ArcGIS (available upon request) |
| R (spatial packages) | Statistical computing with powerful spatial capabilities |
| Python (geospatial libraries) | Versatile programming language with robust GIS support |
| Orfeo Toolbox (OTB) | Advanced remote sensing toolbox |
| SAGA GIS | System for Automated Geoscientific Analyses |
| STATA | Statistical software with spatial econometrics capabilities |
| SAS | Analytics software suite with geospatial procedures |
| MATLAB | Numerical computing environment with mapping toolbox |

### HPC-Enabled GIS Capabilities

- **Big Data Processing**: Analyze terabytes of geospatial data with ease
- **Parallel Computing**: Accelerate analyses using multiple cores
- **Remote Sensing**: Efficiently process satellite imagery and LiDAR data
- **Spatial Statistics**: Perform complex geostatistical analyses at scale
- **Machine Learning for GIS**: Implement advanced AI algorithms for geospatial applications

## Advanced GIS Technologies

### GeoAI: The Fusion of AI and Geospatial Science

GeoAI represents the cutting-edge intersection of artificial intelligence and geospatial analysis. RCC's HPC infrastructure empowers researchers to:

- Uncover hidden spatial patterns in complex datasets
- Develop accurate predictive models using historical geospatial data
- Monitor real-time geospatial data streams for rapid change detection
- Automate feature extraction from satellite imagery and LiDAR data

Our HPC environment supports popular GeoAI frameworks, allowing you to develop and run sophisticated AI models on geospatial data at an unprecedented scale.

### CyberGIS: Unleashing the Power of Cyberinfrastructure

CyberGIS combines advanced cyberinfrastructure with GIS to solve computationally intensive spatial problems. RCC's HPC resources are tailor-made for CyberGIS applications, offering:

- **Scalability**: Process enormous spatial datasets and run complex analytical software
- **Collaboration**: Enable multi-user access to CyberGIS capabilities and shared resources
- **Advanced Tools**: Access cutting-edge spatial analysis techniques optimized for HPC
- **Interdisciplinary Integration**: Seamlessly combine geospatial approaches with other scientific domains

## Accessing GIS on HPC

1. **Request an RCC account** if you don't already have one
2. **Connect to the RCC cluster** via SSH or our user-friendly web portal
3. **Load GIS modules** using the `module load` command (e.g., `module load qgis`)
4. **Execute your GIS workflows** through batch jobs or interactive sessions
